# Components

This folder contains reusable React components.

## Structure

- Keep components modular and reusable
- Each component should have its own file
- Use TypeScript interfaces for props
- Follow naming convention: PascalCase for component files

## Available Components

### Navigation & Layout

- **Navbar.tsx** - Main navigation with responsive menu, scroll effects, and transparent overlay
- **Footer.tsx** - Professional footer with contact info, quick links, and social media
- **ScrollToTop.tsx** - Utility component that scrolls to top on route changes
- **Hero.tsx** - Homepage hero section with background image and CTA

### Content Sections

- **Services.tsx** - Services showcase grid with 4 main service offerings
- **CEOMessage.tsx** - CEO message and company overview section

### Utility Components

- **UnderMaintenance.tsx** - Professional "Under Construction" page with progress indicator and background image

## Example

See `Button.tsx` for an example of a well-structured component with:

- TypeScript interface for props
- Tailwind CSS for styling
- Multiple variants and sizes
- Proper typing
