# Pages

This folder contains page-level components.

Each page represents a distinct route or view in your application.

## Guidelines

- One page per file
- Pages compose smaller components from the `components` folder
- Keep business logic in custom hooks or utilities

## Available Pages

### Home.tsx

Homepage with Hero, Services overview, and CEO Message sections

### About.tsx

Company overview, history, mission, values, and team information

### Services.tsx

Detailed services page with:

- 4 core service offerings (Logistics, Finance, Management Consultancy, Global Operations)
- A to Z Support levels (Country/National, Sector, Company/Project)
- Professional imagery and detailed descriptions
